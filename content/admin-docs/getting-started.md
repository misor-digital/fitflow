# Първи стъпки

## Вход в системата

1. Отидете на `/login` и влезте с вашите служебни данни.
2. Ако акаунтът ви има админ роля, ще видите админ панела след вход.
3. Навигирайте до `/admin` или кликнете върху линка за администрация.

## Първоначална настройка: Създаване на супер администратор

Преди някой да може да достъпи бек офиса, трябва да съществува super admin акаунт.

```bash
npx tsx scripts/seed-super-admin.ts
```

**Необходими environment променливи:**

| Променлива | Описание |
|---|---|
| `NEXT_PUBLIC_SUPABASE_URL` | URL на Supabase проекта |
| `SUPABASE_SECRET_KEY` | Supabase service role ключ |
| `SUPER_ADMIN_EMAIL` | Имейл за super admin акаунта |
| `SUPER_ADMIN_PASSWORD` | Парола (мин. 8 символа, главна буква, малка буква, цифра, специален символ) |
| `SUPER_ADMIN_NAME` | _(по избор)_ Име за показване |

Използвайте `--dry-run` за преглед без създаване на акаунт:

```bash
npx tsx scripts/seed-super-admin.ts --dry-run
```

## Таблото

След вход, `/admin` показва главното табло с:

- **Общо поръчки** — брой на всички поръчки в системата
- **Брой служители** — брой на служителите
- **Вашата роля** — текущата ви админ роля
- **Активни абонаменти** — брой на активните абонаменти
- **MRR** — Месечен рекурентен приход
- **Последно генериране** — статус на последното групово генериране на поръчки

## Навигация в страничния панел

Страничният панел показва само секциите, до които ролята ви има достъп. Вижте [Роли и права](roles-and-permissions) за подробности.

Основни секции:

- **📊 Табло** — `/admin`
- **📦 Поръчки** — `/admin/orders`
- **🔄 Абонаменти** — `/admin/subscriptions`
- **👥 Клиенти** — `/admin/customers`
- **📅 Доставки** — `/admin/delivery`
- **🏷️ Промо кодове** — `/admin/promo`
- **📊 Имейли** — `/admin/emails`
- **👤 Контакти** — `/admin/emails/contacts`
- **📧 Кампании** — `/admin/campaigns`
- **🔀 Кампания за конвертиране** — `/admin/order-subscription-campaign`
- **🚫 Отписани** — `/admin/emails/unsubscribes`
- **👥 Служители** — `/admin/staff`
- **💬 Обратна връзка** — `/admin/feedback`
- **📝 Съдържание** — `/admin/content` _(заготовка)_
- **📈 Анализи** — `/admin/analytics` _(заготовка)_
- **⚙️ Настройки** — `/admin/settings`

В долната част на панела се показва вашето име, роля и линк „Към сайта".
